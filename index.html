Web + Mobile Bundle

الباقة السعر المحتوى المدة
أساسي 60,000 ج موقع + تطبيقين (بسيط) + لوحة تحكم موحدة 60-70 يوم
احترافي 100,000 ج موقع متوسط + تطبيقين متوسطين + مزامنة فورية 80 يوم
متكامل 160,000 ج موقع كبير + تطبيقين معقدين + API كامل 100 يوم
مؤسسي 250,000 ج+ منصة كاملة + BI وتحليلات + دعم 24/7 120-150 يوم

ما يشمله كل باقة:

  • موقع ويب متجاوب (Responsive)
  • تطبيق Android (APK + Play Store)
  • تطبيق iOS (App Store)
  • لوحة تحكم موحدة (Web Dashboard)
  • قاعدة بيانات مشتركة + مزامنة فورية
  • هوية بصرية موحدة
  • توثيق كامل للنظام
  • توفير 30% من تكلفة البناء المنفصل
